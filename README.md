# Rock Paper Scissors

> Best-of-3 Rock Paper Scissors against the computer with live score tracking.

---

## Features
- Best-of-3 match format
- Live score display
- Random computer choice
- Replay support

## Requirements
```
No external dependencies — pure Python 3
```

## Installation
```bash
git clone https://github.com/brainiacweb-tech/rock-paper-scissors
cd rock-paper-scissors
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```
